## August 17 2025
## 0.0.2 app
- Began the compilation and generation of documentation relating to RecipeFirst
	- Included it in the GitHub repository for easier access, sharing and versioning
- Chose markdown and Obsidian as the official format and app for RecipeFirst support documentation.
	- Obsidian is not required but if you open the doc/RecipeFirst directory in Obsidian it will provide a very user-friendly experience. Compatibility with other markdown viewers is likely but not guaranteed
### 0.0.1 www
- Fixes bug causing the footer to overlap the bottom of the content area
- Removes unnecessary columns from List Views
- Formats raw JSON/String Objects in List and Detail views to make them more human readable. This affected the following areas:
	- Recipe Ingredients
	- Recipe Instructions
	- Categories in both List and Detail views
